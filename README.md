<h1>DHCP</h1>

<h2>Description</h2>
This lab will configure DHCP. The DHCP server is on the same subnet as the clients
<br />

<h2>Environments Used </h2>

- <b>Packet Tracer</b>

<h4>Lab Topology:</h4>

The lab consists of 1 router, 1 switch, 1 DHCP server and 3 computers. <br/>
